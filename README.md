# 📌 과제 제출 안내

## 1. 제출 대상
- 해당 **주차 폴더**에서 `frontend` 또는 `backend` 중 하나 선택 (둘 다 제출도 가능)

## 2. 과제 확인 및 작성
- 각 폴더의 **README.md**에 기재된 문제를 확인  
- 문제에 맞춰 프로그램을 작성

## 3. 제출 방식
- 프로그램 **실행 화면 gif 캡처**  
- 또는 프로그램 **전체 코드 파일**

## 4. 파일명 규칙
- 업로드 시 반드시 아래 형식으로 지정

```
이름/파일명
ex) 홍길동/week1.c | 홍길동/week1_실행화면.gif
```

## 5. 그 외 사항
- 정답 예시는 제공 X
