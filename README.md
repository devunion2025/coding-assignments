# 📌 과제 제출 안내

## 1. 분야 선택
- 해당 주차 폴더에서 `frontend` 또는 `backend` 중 하나 선택 (둘 다 제출도 가능)

## 2. 과제 확인 및 작성
- 각 폴더의 **README.md**에 기재된 문제를 확인  
- 문제에 맞춰 프로그램을 작성
- 문제에 지정된 언어가 없다면 **아무 언어로나 작성 가능**

## 3. 결과물
- 프로그램 **실행 화면 gif 캡처**
- 또는 프로그램 **전체 코드 파일**

## 4. 제출 방식
- 업로드 전 반드시 개인 폴더 생성
- `Add file` -> `Create new file` 버튼 클릭
- 아래와 같이 파일명 설정 후 `Commit changes...` 버튼 클릭
```
이름/README.md
```
- 이후 개인 폴더에 작성한 파일 업로드

## 5. 그 외 사항
- **정답 예시 제공 X**
